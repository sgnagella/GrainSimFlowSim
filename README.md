# Initial Configuration for ShearGranularMedia3d Simulation

This file contains the initial configuration of particles for the shear granular media simulation.

## Particle Data
- Each row corresponds to a particle.
- The columns represent the x and y coordinates of the particle centers.

## Lattice Parameters
- Particle radius (R): 1.0
- Overlap distance (δ): 0.008 (0.8% of R)
- Center-to-center spacing: 1.992
- Total number of particles: 256
- Dimensions of rectangular region: 31.88 x 31.88

 - Packing fraction: 0.894

 No. of particles in top row: 16
 No. of particles in bottom row: 16
## Notes
- The top and bottom rows of particles are intended to represent the moving plates in the simulation.
- The configuration is centered around (0,0) for convenience.
